What is a Dynamic array - In Java

31. July 2016 written by Christina Bögh
A dynamic array is an array that is able to handle continuous change - this means you can be adding and removing items from the array at any time.

ArrayList
This type of data structure is an object ( normal arrays are as well) and are in Java called ArrayList. ArrayList makes a list of items and it can hold any element, even null. All the elements in the dynamic array have to be of the same type ( int or String etc.). The ArrayList has something called capacity, which is the size of the array and the capacity increases with the size of items in the array. ("ArrayList (Java Platform SE 7 )," n.d.). The ArrayList is an object the class  java.util library and is implemented as standard in Java. You import it by writing: import java.util.ArrayList; at the top of your code. ("Home and Learn - Array Lists in Java," n.d.). ArrayList has methods for inserting, deleting and searching.

Vector.
Java has a class called Vector, which is basically the same as the ArrayList, but synchronized. Synchronization is important when several threads try to access and update the array at the same time. However, it seems to be the general rule to avoid using Vector and instead use ArrayList with an added synchronization when needed. (Tran, 2016).
In my example, below I implemented such a synchronization by using a method (.synchronizedList) from the import java.util.Collections; List<String> syncList = Collections.synchronizedList(new ArrayList<String>()); (Swartz, 2006). (Pjhanzlik, 2016).

Creation:
When you want to create a new array or ArrayList object of for example type integer this is how you do it and how they are different:
Normal array:
Integer[] ints = new Integer[x]
ArrayList:
List<Integer> ints = new ArrayList<Integer>();
The ArrayList does not need to be initialized with a size like this array is with x. (Swartz, 2006).

Iteration
To go through each item in your ArrayList you can set up something called an Iterator. It is a class also found in the standard library. import java.util.Iterator;

Example:
Create the ArrayList object called syncList like shown below:

public static void main(String[] args) {
List<String> syncList =new ArrayList<>(); // Adding elements to the ArrayList with the .add method.
syncList.add("one");
syncList.add("two");
syncList.add("three");

// when iterating over a synchronized list, we need to synchronize access to the synchronized list.
synchronized (syncList) {
// new Iterator object called iterator. can be used to go through the items in the ArrayList with the method .next. Iterator<String> iterator = syncList.iterator();
  //while loop to iterate through the list.
while (iterator.hasNext()) {
System.out.println("item: " + iterator.next());
}
//searching the array with the .contains method.
System.out.println (syncList.contains("two")); }
}
}

Output:
run:
item: one
item: two
item: three
true
BUILD SUCCESSFUL (total time: 0 seconds)
(Eriksson, n.d.).
As you can see from the output that three Strings were in the ArrayList (“One”, “two”, “three”) and the iterator found and printed them one after the other. The last method (.contains) found that it is indeed true that the String ”two” can be found in the ArrayList, and thus prints true. When to use a dynamic array over a regular array Advantage: If you do not know how many items you want your array to hold, using an ArrayList is probably the best idea, since you can add and remove items to it. Disadvantage: Normal arrays are faster, take a fixed amount of memory and you can make a normal array double- and multi-dimensional, whereas an ArrayList is only one-dimensional.
