How to capitalize the first word in a string - Solution 1



function capitalize(str) {

str = str.toLowerCase().split(' ');
The string str is being converted to all lower case first.
Then the string is split into words.
(' ') means that the program splits at each instance of a space.
Every new word is put into an array(object).
str is no longer a type string.

for (var i = 0; i < str.length; i++) {
Looping through the items of the list until we reach the end ( str.length).

str[i] = str[i].charAt(0).toUpperCase() + str[i].slice(1);
At each iteration [i] the character at position 0, the first character, is capitalized.
Then to this character the rest is added.
slice(1) indicates that the slice that should be added starts at position 1, and since no endslice-position is added it takes the whole bit/word.

} return str.join(' ');
} capitalize("aBout mY houSE");


Output: => 'About My House'
