Svara på följande frågor och ge ett exempel på varje.

Vad är ett statement?

A statmenet is an "instructions" to be "executed" by the web browser. The statements are executed, one by one, in the same order as they are written. Much like sentences, each statmenet is ended with a special sign - in Javascript this is the semicolon ' ; '.

Vad är ett expression?

An expression produces a value and can be written wherever a value is expected, for example as an argument in a function call. According to MDN: An expression is any valid unit of code that resolves to a value.

Vad är en operator?

Comparison and Logical operators are used to test for true or false. if (age < 18) text = "Too young";

Vad är en variabel?

JavaScript variables are containers for storing data values.

Vad är skillnaden mellan String och Number?

One is a string (of characters), the other is a number. Various operators work differently:

alert("1234" + "1234") // "12341234" (string)
alert(1234 + 1234) // 2467 (number)
alert("1234" + 1234) // "12341234" (string)
